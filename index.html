Conversor de Moedas

Este projeto é um conversor de moedas desenvolvido com HTML, CSS e JavaScript. Ele permite ao usuário converter valores entre diferentes moedas de forma simples e rápida. A interface é intuitiva e responsiva, e o sistema realiza os cálculos com base em taxas de câmbio atualizadas. O objetivo do projeto foi praticar manipulação do DOM, eventos em JavaScript e organização de layout com CSS.
